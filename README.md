# SMS Flask App

Simple Flask + Twilio demo for sending SMS messages.

## Local dev

1. Create virtualenv and install dependencies:
   ```bash
   pip install -r requirements.txt

2. Create a .env with TWILIO_ACCOUNT_SID, TWILIO_AUTH_TOKEN, TWILIO_PHONE_NUMBER

3. Run:  python app.py

6. **`requirements.txt`** (freeze your installed packages)

```bash
pip freeze > requirements.txt
